# UniProtKB

| **Website** | [https://www.uniprot.org/](https://www.uniprot.org/)) |
| **License** | [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/) |

<a name="tp1">UniProtKB</a> is a database with information about proteins and their amino acid sequences [<a href="#citeref1">1</a>].

## Entities

* Protein: a <a name="tp2">protein sequence</a>.

## Example queries

The SIB has collected a long list of queries on [this page](https://sib-swiss.github.io/sparql-examples/examples/UniProt/) [<a href="#citeref2">2</a>].
